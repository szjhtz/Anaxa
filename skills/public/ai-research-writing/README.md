This directory is a MedrixFlow-native adaptation of the ideas and prompt
categories from:

https://github.com/Leey21/awesome-ai-research-writing

The upstream repository is not distributed as a MedrixFlow/Claude-style skill
package with a `SKILL.md`, so this wrapper provides a compatible entry point for
MedrixFlow's recursive skill loader.

Source focus:
- academic translation
- polishing
- logic checking
- reviewer-style critique
- title/caption support
- experiment write-up assistance
